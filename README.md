# The Final Clue

The Final Clue is a browser-based detective mystery game.
You investigate cases, interrogate suspects, and accuse who you think is guilty.

## Features

- Multiple mystery cases with unique stories and suspects.
- Interactive accusation flow with result modal (correct or wrong outcome).
- In-game interrogation chat panel for suspect roleplay.
- Express backend proxy for Gemini API calls.

## Project Structure

- index.html: Frontend UI, styles, game logic, and chat interactions.
- server.js: Express server and chat API endpoint.
- pics/: Game and UI images.
- package.json: Scripts and dependencies.

## Requirements

- Node.js 18 or later recommended.
- A valid Gemini API key.

## Installation

1. Open a terminal in the project folder.
2. Install dependencies:

	 npm install

## Environment Variables

Create a .env file in the project root with:

GEMINI_API_KEY=your_real_key_here

Important:
- The backend reads GEMINI_API_KEY from .env.
- The frontend currently also checks a key constant in index.html before sending chat requests.
	Set this line in index.html to your key as well:

const GEMINI_API_KEY = "your_real_key_here";

## Run Locally

Start the app:

npm start

Server default URL:

http://localhost:3000

Open that URL in your browser.

## How to Play

1. Click Play Game.
2. Read the case and select a suspect.
3. Click Accuse.
4. Use the chat button to open interrogation and question suspects.

## API Endpoint

Backend endpoint used by frontend chat:

POST /chat

Request body:

- message: user question text
- systemPrompt: roleplay prompt generated by frontend

## Troubleshooting

- Buttons not working:
	Reload the page with cache disabled and verify JavaScript is enabled.

- Chat says key is missing:
	Ensure GEMINI_API_KEY is set in .env and in index.html.

- Cannot GET /:
	Make sure you started the app with npm start from the project root.

- Chat connection issues:
	Confirm server is running and accessible at http://localhost:3000.
